SkyTime is a GPS/PPS disciplined Stratum-1 NTP Server built on the Waveshare ESP32-P4-ETH PoE platform.

Features
GPS disciplined UTC time
PPS timing validation
Stratum-1 NTP service
Ethernet networking
DHCP or Static IP operation
LCD local status display
Dashboard web interface
System status page
Read-only Event Log viewer
Read-only NTP Log viewer
Identity configuration
Network configuration
Web initiated reboot
Configuration persistence
Hardware
Board: Waveshare ESP32-P4-WiFi-PoE-ETH PoE Display: ST7789 240x320 SPI GPS: ATGM336H PPS: GPIO46 (Rising Edge) Button: (Active LOW)

Configuration
Default Hostname: SkyTime

Default Static Network: IP Address : 192.168.0.123 Subnet Mask: 255.255.255.0 Gateway : 192.168.0.1 DNS Server : 192.168.0.1
